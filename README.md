# Meta Full Stack Journey

Repositório de estudo e prática da especialização **Meta Full Stack Developer**.

O objetivo deste projeto é centralizar anotações, sínteses de aulas e exercícios práticos desenvolvidos ao longo do curso, com foco em aprendizado progressivo e documentação útil para revisão futura.

## Escopo

Este workspace reúne dois tipos principais de material:

- notas de estudo em Markdown organizadas por curso e módulo
- projetos práticos e exercícios implementados durante a trilha

O conteúdo é estruturado para funcionar tanto como registro de progresso quanto como base de revisão técnica.

## Estrutura

```text
meta-fullstack-journey/
  notes/
    course-5-react-basics/
  course-5-react-basics/
    components-react-onde-residem/
```

### Pastas principais

- `notes/`: anotações, resumos e materiais de revisão do curso
- `course-5-react-basics/`: exercícios, exemplos e projetos relacionados ao módulo de React

## O que você encontra aqui

- notas organizadas por assunto
- resumos reescritos a partir das aulas
- exemplos práticos em React
- pequenos projetos para consolidar fundamentos

## Como usar

### Notas

As notas podem ser lidas diretamente em Markdown ou abertas em um editor compatível com Obsidian.

### Projetos React

Cada projeto prático pode ter sua própria configuração e dependências. Quando houver um `package.json` no diretório do exercício:

```bash
cd course-5-react-basics/components-react-onde-residem
npm install
npm run dev
```

## Organização

O repositório segue uma abordagem simples:

- separar estudo teórico de prática
- manter nomes de pastas alinhados com os módulos do curso
- documentar os conceitos de forma reutilizável para revisão posterior

## Status

Projeto em evolução contínua, acompanhado conforme o avanço na especialização.
